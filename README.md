# hackerrank-sql-postgresql
Colección de ejercicios SQL de HackerRank resueltos en PostgreSQL

---

Siguiente paso:

1. Ejecuta en tu BD (conectado a la base que usarás para los retos):
   - `schema_postgres.sql`
   - `data_sample.sql`
2. Corre `solution_postgres.sql` y valida el resultado.
3. Haz `git add .`, `git commit`, `git push` para subir este primer problema.
